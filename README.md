
# HomeForRent

A web based application where user looking for room can rent a room and user who wants to put their home on rent can advertise their home.




## Technology used
- Java 8
- Spring Boot
- JWT
- HTML, CSS, JavaScript
- Thymeleaf
- MongoDB
- Hibernate
## Deployment

Clone project from GitHub

```bash
  git clone repo_link;
```
Install jdk and maven to run

```bash
  install jdk;
  indtall maven;
```
Run project

```bash
  mvn spring-boot:run
```




## Features

- Chat Feature
- Contract Signing
- Map Locations
- Current Landlord/Tenant
- Secure Payment
